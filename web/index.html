Everything you need, nothing you don't

Built for developers who want Claude Code power with Discord convenience.

Multi-Session

Run up to 10 concurrent Claude Code sessions. Each Discord channel gets its own independent agent with isolated context.

Skill System

Inject specialized skills per channel -- SEO expertise, design patterns, coding standards. Each agent becomes a domain specialist.

Cost Tracking

Per-channel cost breakdown with daily and weekly reports. Know exactly how much each project consumes. Use !cost anytime.

Memory Compaction

Automatic conversation summarization after N messages. Long-running projects stay coherent without context window overflow.

Cross-Channel Routing

Send commands to any channel from a central hub. Orchestrate multiple agents from #general without switching channels.

Auto-Update

Type !update in Discord and DisClawd pulls the latest code, rebuilds, and restarts with zero downtime. No SSH needed.

Up and running in minutes

Four steps from zero to a multi-agent Discord workspace.

01

Create channels

Create Discord channels for your projects. Each channel becomes an independent agent workspace with its own context and history.

02

Configure per-channel

Set the model, inject skills, and point to the right working directory. All via a simple JSON config or Discord commands.

03

Chat normally

Send messages like you would to any Discord bot. DisClawd spawns Claude Code sessions behind the scenes and streams responses live.

04

Monitor everything

Live streaming responses, todo tracking, cost monitoring. Full visibility into every agent's work from your phone or desktop.

Ready to try? It takes 2 minutes.

Clone, configure, run. That's it.

terminal 
# Clone the repo
git clone https://github.com/XavierKain/disclawd.git
cd disclawd

# Install dependencies
bun install

# Configure your bot token
cp .env.example .env
nano .env

# Start DisClawd
bun start
